PROBLEM STATEMENT: Simulating the rising of warm, moist air to form convective clouds and rain. 
Also studying how humidity and aerosols affect when condensation begins and how the cloud develops.

I used computer models like WRF and CM1 to simulate how clouds form and grow.
Python tools helped me analyse temperature, wind, and rainfall and create visual simulations from the data.
